== Description ==

This plugin is a simple newsletter plugin. It can show opt-in form, save opt-in email and name, and send emails to your opt-in list. It also have import and export email data ability. You can comment and ask about the plugin <a href="http://smallwebsitehost.com/doc/stat.zip" target="_blank" title="wordpress stat plugin">here</a>.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can see the stat at "settings" -> "stat" menu.
